## 简介
在正式的下载和安装之前，让我们先来了解一下[**IntelliJ IDEA**](https://www.jetbrains.com/zh-cn/idea/ "IDEA官网，点击跳转")这款软件，
其为[JetBrains](https://www.jetbrains.com/zh-cn/ "JetBrains官网，点击跳转")
开发的集成开发环境（IDE），主要用于**Java**开发，但也支持多种编程语言，如 Kotlin、Scala、Groovy、JavaScript、TypeScript 等。它是市场上最受欢迎的 Java IDE 之一，
相较于[**Eclipse IDE**](https://www.eclipse.org/ide/)，它拥有**更现代化的界面**、**更智能的代码补全**、**更丰富的插件扩展**等，目前广泛应用于企业开发和开源项目中。

*IntelliJ IDEA*分为社区版（IntelliJ IDEA Community）与专业版（IntelliJ IDEA Ultimate）

### 版本区别

<Accordions type="single">
  <Accordion title="社区版 (Community Edition)">
    免费开源软件，面向*Java*和*Kotlin*爱好者的**IDE**，满足日常个人编程使用。
  </Accordion>
  
  <Accordion title="专业版 (Ultimate Edition)">
    付费版本，提供企业级Java开发、Spring框架、前端技术、数据库工具等全栈开发支持，
    功能相比社区版更强大，面对企业级开发更专业。
  </Accordion>
</Accordions>

### 自行下载操作流程
首先我们前往[JetBrains官网](https://www.jetbrains.com/zh-cn/ "JetBrains官网，点击跳转")，
在**顶部导航栏**选择**开发者工具**，点击**IntelliJ IDEA**
![idea-i-2](images/idea/idea-i-2.png)
进入下载界面后选择合适的操作系统版本，点击下拉框可选择下载形式，Windows推荐直接下载.exe。
![idea-i-3](images/idea/idea-i-3.png)
**点击下载**，即可开始下载，完成后打开文件。
![idea-i-4](images/idea/idea-i-4.png)
## 安装
**点击运行**
![idea-i-5](images/idea/idea-i-5.png)
**点击浏览**，选择适当的文件地址下载，**不建议默认放置到C盘**。**点击下一步**
![idea-i-6](images/idea/idea-i-6.png)
创建桌面快捷方式，更新环境变量，创建关联。此间设置在后续皆可再作调整，并非不可更改。
![idea-i-7](images/idea/idea-i-7.png)
**点击安装**等待完成即可
![idea-i-8](images/idea/idea-i-8.png)
**点击完成**，允许公用网络和专用网络访问，开启你的Java学习之旅。
![idea-i-9](images/idea/idea-i-9.png)
## 许可证激活
选择中文语言包，地区中国大陆，是否选择英文的看个人习惯。
![idea-l-1](images/idea/idea-l-1.png)
选择不发送，防止垃圾信息邮件轰炸
![idea-l-2](images/idea/idea-l-2.png)
选择跳过导入
![idea-l-3](images/idea/idea-l-3.png)
点击左下角设置图标
![idea-l-4](images/idea/idea-l-4.png)
选择管理订阅
![idea-l-5](images/idea/idea-l-5.png)
在教育许可证申请通过后，此界面选择登录JetBrains账号。
![idea-l-6](images/idea/idea-l-6.png)
在浏览器登录账号，授权成功后关闭页面，回到IDEA。
![idea-l-7](images/idea/idea-l-7.png)
**点击激活**
![idea-l-8](images/idea/idea-l-8.png)

现在你已经完成了IDEA专业版的下载和安装，接下来我们来创建一个简单的Java项目并运行。

## 简单的项目创建和运行

### 如何创建一个新项目

**点击新建项目**
![idea-n-1](images/idea/idea-n-1.png)
**项目名自拟**主要命名规则，选择合适的位置存放代码，导入JDK，创建即可。
![idea-n-2](images/idea/idea-n-2.png)
**右键src源码**，新建，Java类，注意命名规则，回车。
![idea-n-3](images/idea/idea-n-3.png)
### 项目运行
右上角运行，绿色对号表示代码无误可运行，
黄色感叹号表示弱警告但仍可运行，红色感叹号表示严重警告，运行报错，需调试。
运行结果在下方命令行中显示。
![idea-n-4](images/idea/idea-n-4.png)
